# AppDaemon приложения

## Общая конфигурация

### Основные файлы
- `appdaemon.yaml` - Основная конфигурация
- `apps.yaml` - Конфигурация приложений
- `globals.py` - Глобальные переменные и функции
- `automation.py` - Базовые функции автоматизации

### Глобальные модули
- `globals` - Общие функции и переменные
- `automation` - Автоматизация
- `notify` - Уведомления

## Структура приложений

### Освещение (light/)

#### lightcontrol.py
**Функционал**: Основное управление освещением
**Особенности**: 
- Автоматическое управление по времени
- Интеграция с датчиками движения
- Сценарии освещения

#### lightcontrol_kitchenset.py
**Функционал**: Управление освещением кухни
**Особенности**:
- Автоматическое включение при входе
- Режимы освещения для разных задач
- Интеграция с датчиками присутствия

#### lightcontrol_bath.py
**Функционал**: Управление освещением ванной
**Особенности**:
- Автоматическое включение при входе
- Таймеры для экономии энергии
- Интеграция с датчиками движения

#### lightcontrol_hall.py
**Функционал**: Управление освещением прихожей
**Особенности**:
- Автоматическое включение при входе в дом
- Режим дежурного освещения
- Интеграция с системой безопасности

#### light_mainroom.py
**Функционал**: Управление освещением гостиной
**Особенности**:
- Сценарии для просмотра ТВ
- Автоматическое затемнение вечером
- Интеграция с датчиками присутствия

#### light_balcony.py
**Функционал**: Управление освещением балкона
**Особенности**:
- Автоматическое включение при открытии двери
- Таймеры для экономии энергии
- Интеграция с датчиками движения

#### lightbutton.py
**Функционал**: Управление кнопками освещения
**Особенности**:
- Обработка нажатий на физические кнопки
- Двойные и длинные нажатия
- Интеграция с системой автоматизации

### Безопасность (security/)

#### Система сигнализации
**Функционал**:
- Мониторинг состояния датчиков
- Автоматическая постановка/снятие с охраны
- Интеграция с системой уведомлений

### Присутствие (presence/)

#### Датчики присутствия
**Функционал**:
- Определение присутствия в комнатах
- Автоматическое управление освещением
- Интеграция с системой безопасности

### Замки (locks/)

#### Управление замками
**Функционал**:
- Автоматическое запирание/отпирание
- Интеграция с системой безопасности
- Уведомления о состоянии замков

### Домофоны (doorbells/)

#### Система домофонов
**Функционал**:
- Обработка вызовов домофона
- Автоматические уведомления
- Интеграция с системой безопасности

### Энергетика (power/)

#### Мониторинг энергопотребления
**Функционал**:
- Отслеживание потребления электроэнергии
- Автоматическое отключение неиспользуемых устройств
- Анализ энергоэффективности

### Климат (climate/)

#### Управление климатом
**Функционал**:
- Автоматическое управление кондиционерами
- Поддержание комфортной температуры
- Энергосберегающие режимы

### Сенсоры (sensors/)

#### Обработка данных сенсоров
**Функционал**:
- Агрегация данных с различных сенсоров
- Автоматические уведомления при превышении порогов
- Логирование и анализ данных

### Группы (groups/)

#### Управление группами устройств
**Функционал**:
- Группировка устройств по комнатам
- Массовые операции с устройствами
- Сценарии для групп устройств

### Алиса (alice/)

#### Интеграция с Яндекс.Алиса
**Функционал**:
- Голосовое управление устройствами
- Обработка голосовых команд
- Интеграция с системой автоматизации

### Уведомления (notifiers/)

#### Система уведомлений
**Функционал**:
- Отправка уведомлений в различные каналы
- Автоматические уведомления о событиях
- Настройка приоритетов уведомлений

## Функционал

### Автоматизация
- **Временные триггеры**: Автоматические действия по расписанию
- **Событийные триггеры**: Реакция на изменения состояния устройств
- **Условная логика**: Сложные сценарии с множественными условиями

### Интеграции
- **Home Assistant**: Основная интеграция через API
- **MQTT**: Обмен сообщениями с другими сервисами
- **HTTP**: Внешние API и веб-хуки

### Мониторинг
- **Логирование**: Детальное логирование всех операций
- **Метрики**: Сбор статистики работы приложений
- **Алерты**: Уведомления о критических событиях

### Безопасность
- **Аутентификация**: Проверка прав доступа
- **Шифрование**: Защита конфиденциальных данных
- **Аудит**: Логирование всех действий пользователей

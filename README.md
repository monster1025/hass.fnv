# Home Assistant FNV - Документация

Репозиторий содержит конфигурацию умного дома на базе Home Assistant с интеграцией различных сервисов и устройств.

## Структура документации

- [Docker контейнеры и настройки](docs/docker.md) - Описание всех Docker сервисов и их конфигурации
- [Home Assistant конфигурация](docs/home-assistant.md) - Настройки Home Assistant, устройства и интеграции
- [AppDaemon приложения](docs/appdaemon.md) - Скрипты, дашборды и функционал AppDaemon

## Быстрый старт

1. Склонируйте репозиторий
2. Настройте переменные окружения в соответствующих `.env` файлах
3. Запустите `docker-compose up -d`

## Основные компоненты

- **Home Assistant** - Центральная система умного дома
- **AppDaemon** - Автоматизация и скрипты
- **ESPHome** - Управление ESP устройствами
- **Zigbee2MQTT** - Zigbee устройства
- **Node-RED** - Визуальное программирование
- **Mosquitto** - MQTT брокер
- **PostgreSQL** - База данных
- **Nginx** - Веб-сервер с SSL
- **Matter Bridge** - Интеграция с Matter устройствами
